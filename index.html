本地对外发布指南

方法一：使用内网穿透工具（推荐）

步骤：

  1. 下载内网穿透工具（如 ngrok、frp、natapp 等）
  2. 注册账号并获取免费隧道
  3. 配置工具指向本地端口（如 8080）
  4. 启动工具，获得公网访问地址
  5. 将地址分享给外部用户

推荐工具：

  • Ngrok：https://ngrok.com/
  • Sunny-Ngrok：https://www.ngrok.cc/
  • Frp：https://github.com/fatedier/frp

方法二：使用 Python 简单服务器

步骤：

  1. 将本HTML文件保存为 index.html
  2. 打开命令提示符，进入文件所在目录
  3. 运行命令：python -m http.server 8080
  4. 配合内网穿透工具使用

方法三：使用 Node.js 静态服务器

步骤：

  1. 安装 Node.js
  2. 安装 http-server：npm install -g http-server
  3. 进入文件目录运行：http-server -p 8080
  4. 配合内网穿透工具使用

方法四：使用云存储服务

步骤：

  1. 注册 GitHub Pages、Vercel 或 Netlify 账号
  2. 上传HTML文件到平台
  3. 获得免费域名访问地址
  4. 注意：部分平台可能需要简单配置
注意事项

• 数据存储在浏览器本地，不同用户数据独立

• 如需统一数据管理，建议部署后端服务

• 确保网络环境安全，避免敏感信息泄露
